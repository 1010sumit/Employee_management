# Employee-Management-System-
Designed and developed Employee Management System using J2EE, Servlets, MySQL and Java as part of a dynamic web based project. Created functionalities for Admin, Managers and Employees roles. Admin can run payroll for all employees and managers based on leaves and bonus they received on yearly basis. Super manager handles directories hierarchy between different managers based on permission assigned to each directory and each documents. Each employee to be assigned under different managers and they can upload any documents to his managerâ€™s directory based on the permissions assigned to those directories.

## Getting Started
* Java v1.7+
* MySQL
* Spring Tools Suite

## Prerequisites
Downlaod and add the following jar in the project build path
* mysql-connector-java-5.1.40-bin.jar
* servlet-api-2.5.jar

## Run
* Select Project -> RunOnServer -> Select the exisiting server(Jetty/Pivotal) -> Select EMS-jar file -> Click Finish
* Project runs on port 8080.
* http://localhost:8080/EMS/
